# UCLA_admissions
nccucs 1072 data science final project

### 程式需求
- Python 3.7
- mongodb
- node

### 事前準備 -start mongodb
- macOS `brew services start mongodb-community` 
- win `mongod --dbpath <your path>` (local mongodb setup on port 27017)

### start server
- `npm start`
